# DayCost (日均成本计算器)

**DayCost** 是一款极简而强大的个人财务资产追踪应用。不同于传统的记账本（仅记录支出的瞬间），DayCost 的核心哲学是**“微积分摊销”**。它帮助你直观地审视你买来的电子产品、大件资产、甚至是零碎配件，每一天究竟在无形中耗费你多少钱。

通过引入生命周期（损坏、回血）以及嵌套组合机制，DayCost 能为你呈现极其真实且扎心的“电子设备日均财务报表”。

---

## ✨ 核心特性 / Core Features

### 1. 核心推演算法 (Daily Cost Calculation)
底层使用严格的流逝时间进行摊薄分摊：
- **日均成本追踪**：购买总价 ÷ 实际拥有天数 = 日均耗血率（￥/天）。
- 伴随着时间的流逝，只要物品还在服役且未损坏，它的日均成本就会被每天一点点摊平拉低。

### 2. 生命周期与死账锁定 (Item Lifecycle Management)
物品不只是被“买来”，它还会经历生老病死。DayCost 为物品提供了三种动态状态：
- 🟢 **使用中 (Active)**：正常计费，时间与现实世界同步流逝。
- 🔴 **已损坏 (Broken)**：一旦标记损坏并设定损坏日，时间将被永远冻结。沉没成本不再摊薄，**锁定最糟糕的日均历史效率作为永久死账**。
- 🟡 **已售出/回血 (Sold/Resold)**：填入回血金额（二手卖出的残值），系统会自动重算「净花费」，用最真实的剩余成本去计算这件物品陪伴你的日均开销。

### 3. 树形组合与聚合透视 (Hierarchical Cost Tracking)
完美支持“套娃装机”等复杂场景（比如：你给一台旧电脑不断更换显卡、加硬盘）：
- **父子层级关系**：可以将“显卡”、“风扇”等作为子零件归属到“电脑”这个组合分类下。
- **并行摊薄与合并账单**：子零件享有自己的独立时间线（比如显卡才用了50天，机箱已经用了3年）。而“电脑”作为父级，它的卡片会**实时自动聚合其腹中所有子零件的总金额与日均耗费**。
- **防套娃机制**：系统在后端级联逻辑中限制了单层树状结构（禁止一级以上的递归循环），保证数据链路稳定和计算性能。

### 4. 财务高管级仪表盘 (Global Stats HUD)
在屏幕左上角常驻毛玻璃态的情报核心网，提供全盘上帝视角：
- **资金总投入**：你这辈子在这个图谱里到底砸进去了多少钱。
- **当前每日总耗**：你今天不仅需要呼吸，仅仅为了维持这些电子设备的生命线，每天还在无形中蒸发多少钱。
- **战损库存统计**：实时统计所有物品的 服役中 / 重大损坏 / 闲置回血 数量总计。

### 5. 高级检索引擎与树洞穿透 (Advanced Deep Filtering)
- **多维组合过滤**：支持基于 状态（使用中/已损坏/已售出） + 多维度排序（按成本、按总价、按存活天数顺逆排）。
- **深层靶向搜索**：全局无感模糊查询。当搜索零件（即使该零件被隐藏在电脑分类下），系统也会穿透找到父级并单独剥离展现该零件，且**保证父级的总体财务聚合账面不变，杜绝数据失真**。

### 6. 极客美学界面 (Premium Glassmorphism UI)
彻底抛弃廉价的 Bootstrap 风格，全程重写沉浸式高质感前端：
- **全局暗黑主题** (Dark Mode)。
- 吸星大法的 **Glassmorphism (毛玻璃) 玻璃态组件**渲染。
- 高精度的自定义弹窗（放弃了丑陋的浏览器原生 alert/confirm 等阻断式弹窗），以及带有微小阻尼感和小箭头的定制下拉框与原生覆盖的日期选择器。

## 🚀 技术栈 / Tech Stack

本项目坚持轻量级、无臃肿框架的纯粹实现：
- **前端 (Frontend)**: 
  - `HTML5` / `Vanilla JavaScript (ES6+)` 
  - 纯手写 `CSS3` (包含响应式 Grid/Flex 布局和 CSS 微动画)。
- **后端 (Backend)**:
  - `Node.js` + `Express.js`。
  - RESTful API 架构支撑前端 CRUD 请求。
- **数据库 (Database)**: 
  - `SQLite3`。轻量级嵌入式数据库，极低负担部署。
- **验证机制 (Auth)**:
  - `JWT (JSON Web Tokens)` 登录与接口鉴权。
  - `bcrypt` 密码加密哈希存储。

## 📦 部署与运行 / Quick Start

本系统为零配置即可运行的设计：

1. **安装环境依赖**：
   确保你的机器上安装了 `Node.js`，然后进入项目根目录：
   ```bash
   npm install express sqlite3 bcrypt jsonwebtoken
   ```

2. **启动服务**：
   ```bash
   node server.js
   ```

3. **打开你的日均成本计算器**：
   在浏览器中访问：`http://localhost:3000`
   > *首次使用请点击「注册」设置你的日均成本管理员账号*。
